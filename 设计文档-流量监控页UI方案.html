🎨 配色方案 — 多巴胺调色板

图表使用独立的多巴胺配色，不受 ocgt 主题色（accent color）限制。让数据展示更鲜明、更易区分。

Chart.js 全局默认色板覆盖，不继承 CSS 变量 --accent。 每个数据集自动分配不同色相，确保视觉区分度。

🧩 交互行为

全局

  • 所有图表 hover 显示 tooltip（数值、占比、时间）
  • 图例可点击切换显示/隐藏系列
  • 时间范围切换带动画过渡（fade + slide）
  • 响应式布局：大屏并排，小屏堆叠

高级

  • 点击环形图扇区 → 下钻到该模型请求列表
  • 表格表头点击排序（时间升/降、Token 大小等）
  • 搜索框实时过滤（debounce 300ms）
  • 导出按钮：仅当前筛选结果 or 全部数据
