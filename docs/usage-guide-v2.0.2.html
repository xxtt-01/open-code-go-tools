概述 · Overview

ocgt (OpenCode Go Tools) 是专为 Claude Code 与 OpenCode Go 打造的原生桌面控制中心与本地代理。 内建 Anthropic ↔ OpenAI 协议互转引擎，提供中英双语 GUI， 支持一键终端启动、流量监控、套餐额度看板。

v2.0.2 增加了 6 项新功能，修复了 10+ 个问题， 完成了定价系统、Token 追踪、流量监控等核心模块的迭代升级。

功能 · Features

v2.0.2 提供 5 个核心面板，快捷键一键切换：

系统状态 Dashboard

监听端口、上游 API、API Key、默认模型。集成状态一目了然。

Ctrl+1

配置管理 Settings

API Key、Profile、模型映射、限流、环境变量、日志。修改即检测。

Ctrl+2

快速连接 Quick Connect

一键终端 + 4 种集成 (CLI/VS Code/settings/Desktop App)。

Ctrl+3

流量雷达 Traffic Radar

多巴胺仪表盘——统计卡片、折线图 (自适应粒度)、环形图、模型表。

Ctrl+4

流量明细 Traffic Detail

10 字段全维度表格。分页 + 三维筛选 + CSV 导出 + 一键清除。

Ctrl+5

亮点

多巴胺配色

15 色高饱和度调色板，自适应亮/暗主题。5 种预设 + 自定义色相。

自适应粒度

≤2 天→小时，≤90 天→天，>90 天→周。tension 0.4 平滑曲线。

重试 + 熔断

同模型 5 次指数退避重试。30 秒熔断，503 快速返回。

额度看板

Rolling / Weekly / Monthly 三级额度进度条实时展示。

快速开始 · Quick Start

1

下载并运行

build/bin/ocgt.exe 直接运行，系统栏常驻。

2

配置 API 密钥

Ctrl+2 → 输入 sk-... 密钥 → 保存。

3

启动终端

Ctrl+3 → 选择 Shell → "打开临时终端"。输入 claude 开始。

4

监控流量

Ctrl+4 雷达仪表盘 · Ctrl+5 请求明细。

默认监听 127.0.0.1:8787 · 首次启动允许防火墙通信

面板导航 · Panel Navigation

系统状态 Ctrl+1

6 个核心指标卡片：监听地址、上游 API、超时、API Key、Profile、默认模型。下方显示 CLI / VS Code / Claude Desktop 配置状态。

配置管理 Ctrl+2

API 代理配置 → 网络限流 → 模型策略 → 高级环境变量 → 日志存储 → 偏好设置。保存即热重载。

快速连接 Ctrl+3

6 张集成卡片：一键修复、临时终端、CLI (全局)、VS Code、Claude Code settings、Claude Desktop App (3P Profile)。

流量雷达 Ctrl+4

统计卡片 → Token 趋势折线图 → 请求量折线图 → 模型环形图 → 模型明细表 → 客户端来源表。时间范围切换 今日 / 7日 / 30日 / 全部。

流量明细 Ctrl+5

10 字段全维度表格：时间 / 模型 / 状态 / Input / Output / Cache / Total / 延迟 / 来源 / 错误。三维筛选 + 分页 + CSV 导出 + 一键清除。

技巧 · Tips & Tricks

快捷键

Ctrl+1–5 切换面板。Esc 关闭模态框。

同步模型

配置管理 → "同步上游模型" 自动拉取最新模型。

永久保存

日志保留天数设 0 = 永久。

CSV 导出

含 BOM 头，Excel 直接打开中文不乱码。

自定义色

偏好设置输入 0–360 色相值，任意颜色。

端口恢复

启动自动释放旧进程端口，无需手动 taskkill。
